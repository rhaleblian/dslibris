Adding Books

Place EPUB files into the Book directory.

Browsing Books

  • UP/DOWN: Select a book
  • START/A/RIGHT: Open the selected book

Tap on a book entry to open it.

Reading a Book

  • RIGHT: Next page
  • LEFT: Previous page
  • UP: Next bookmark
  • DOWN: Previous bookmark
  • A: Next page
  • B: Previous page
  • X: Invert colors
  • L: Previous page
  • R: Next page
  • START: Return to the book browser
  • SELECT: Add a bookmark

Tap on the left and right sides of the touchscreen to move through pages.

Adding Fonts

Add TrueType TTF files to the Font directory. You can then select them using the settings button in the book browser.
