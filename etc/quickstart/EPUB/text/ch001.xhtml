Adding Books

Place EPUB files into the Book directory.

Browsing Books

  • UP/DOWN/SHOULDER: Select a book
  • A/RIGHT/START: Open the selected book
  • SELECT/Y: Enter and leave settings

Reading a Book

Tap on the left and right sides of the touchscreen to move through pages.

  • RIGHT: Next page
  • LEFT: Previous page
  • UP: Next bookmark
  • DOWN: Previous bookmark
  • A: Next page
  • B: Previous page
  • X: Invert colors
  • Y: Add a bookmark
  • SHOULDER: Next or previous page
  • SELECT: Add a bookmark
  • START: Return to the library

Adding Fonts

Add TrueType TTF files to the Font directory. You can then select them in the settings screen.
