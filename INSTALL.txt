
INSTALLATION

1. If needed, DLDI patch dslibris.nds to work for your cart, e.g.

	  dlditool etc/dldi/yourcart.dldi dslibris.nds

   dslibris releases are already patched for R4 carts.

2. Copy all files and folders in this distribution .zip
   to the root folder of your DS media.
   Using a subfolder does not work!

3. Put any EPUB you have into the 'Book' folder.
   The filenames should have an .epub extension.

4. Run dslibris.nds on the DS.


TROUBLESHOOTING

Look for the file `dslibris.log` on your media root.
Send this file to developers when you encounter a problem.
Contact us via the dslibris Issues section on GitHub.
